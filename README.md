<h1>🧠 Neural Network from Scratch (Python)</h1>

Простая реализация многослойной нейронной сети с нуля (без библиотек ML) на Python.

Проект демонстрирует:

прямое распространение (forward pass)
обратное распространение ошибки (backpropagation)
использование momentum
обучение на небольшом наборе данных
<h1>📌 Архитектура сети</h1>
Входной слой (2 нейрона)
        ↓
Скрытый слой 1 (3 нейрона)
        ↓
Скрытый слой 2 (2 нейрона)
        ↓
Выходной слой (1 нейрон)
<h1>⚙️ Параметры сети</h1>
Параметр	Значение	Описание
numInpNeurons	2	входные признаки
numHidNeurons1	3	первый скрытый слой
numHidNeurons2	2	второй скрытый слой
numOutNeurons	1	выход
rateTraining	0.1	скорость обучения
y	0.1	момент (momentum)
learningLimit	85%	критерий остановки
<h1>📊 Данные</h1>
dataSensor = [
    [0.11, 0.32],
    [0.68, 0.89],
    ...
]

expectedOutput = [0.2, 0.7, ...]

Сеть обучается сопоставлять входные векторы с ожидаемыми значениями.

<h1>🧮 Функция активации</h1>

Используется сигмоида:

\sigma(x)=\frac{1}{1+e^{-x}}

Она:

сглаживает значения
переводит их в диапазон (0, 1)
удобна для градиентного спуска
<h1>🔁 Как работает обучение</h1>
1. Forward Pass

Данные проходят через все слои:

Input → Hidden1 → Hidden2 → Output

Каждый нейрон:

sum = Σ (input * weight)
output = activation(sum)
2. Backpropagation

Ошибка на выходе:

error = expected - actual

Затем ошибка распространяется назад:

сначала выходной слой
затем скрытые слои
3. Обновление весов

#Формула обновления:

\Delta w = \eta \cdot error \cdot input + \gamma \cdot \Delta w_{prev}

где:

η — learning rate
γ — momentum
#🚀 Особенности реализации</h1>

✔ Поддержка momentum
✔ Два скрытых слоя
✔ Ручное управление весами
✔ Подробный вывод процесса обучения

<h1>📈 Критерий остановки</h1>

Обучение заканчивается, когда:

networkProgress >= learningLimit

Где networkProgress — точность по последнему примеру.
